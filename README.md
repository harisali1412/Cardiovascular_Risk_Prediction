# AI Feature Extraction and Classification

## Project Overview:
This project demonstrates the application of feature extraction techniques on the **Iris dataset** using **PCA** and **RFE** to reduce dimensionality and select important features. The project includes:
- Feature extraction code
- Dataset and preprocessing steps
- Model training and performance evaluation
- Results and visualizations

## Steps:
1. **Feature Extraction**:
   - PCA for dimensionality reduction
   - RFE for feature selection
   
2. **Model Training**:
   - Applied various classifiers and evaluated performance metrics.

3. **Results**:
   - Accuracy, precision, recall, and other performance measures are reported.

## How to Run:
1. Clone this repository.
2. Install dependencies using `pip install -r requirements.txt`.
3. Run the Jupyter notebooks or Python scripts to see the results.

## Files:
- `/code`: Contains the Python scripts and Jupyter notebooks.
- `/data`: Contains the dataset files.
- `/results`: Contains output files such as accuracy results and classification reports.
- `/images`: Contains visualizations (plots) generated during the project.
